# Аналитика поставщиков в Power BI

Дашборд для оценки эффективности поставщиков — смотрим кто задерживает, 
кто везёт больше и у кого лучший рейтинг.

## Что внутри

- Объём заказов по каждому поставщику
- Процент задержек — считала через DAX-меру
- Рейтинг поставщиков
- Фильтр по категории товара — можно смотреть только электронику или одежду

## Новое по сравнению с Проектом 1

- Две связанные таблицы (поставщики + заказы)
- DAX-меры: COUNTROWS, FILTER, DIVIDE, AVERAGE
- Срез для интерактивной фильтрации

## Инструменты

Power BI Desktop, DAX, CSV
